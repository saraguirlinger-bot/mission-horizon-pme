Unités d'œuvre

Le choix de l'unité d'œuvre détermine comment les charges indirectes sont réparties entre les produits.

Choix de méthode
Pour chaque centre d'analyse, choisissez l'unité d'œuvre la plus pertinente.
Règle : L'unité d'œuvre doit refléter le facteur qui génère réellement les coûts du centre. Elle sert de base pour calculer le coût unitaire d'UO qui sera ensuite imputé aux produits.
Pour choisir : posez-vous la question — "Qu'est-ce qui fait varier les coûts de ce centre ?" Si c'est le temps passé sur une machine → heures machine. Si c'est le travail humain → heures MOD.

✅ Correction — Unités d'œuvre

  • Découpe → heures machine : le coût de ce centre est lié au temps de fonctionnement des machines (amortissement, énergie, usure).
  • Assemblage → heures de MOD : opération manuelle — c'est le temps des opérateurs qui génère les coûts.
  • Stockage → mouvements de stock : chaque entrée/sortie mobilise des ressources (manutention, gestion, risque de rupture).
  • Distribution → livraisons : chaque livraison a un coût fixe propre (camion, chauffeur) indépendant du montant de la commande.
